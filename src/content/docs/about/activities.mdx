## システム開発

### 作画王グランプリ エントリー審査システム

作画王グランプリのエントリー～審査までを管理する審査システムのフロントエンド・バックエンド・運用を担当しました。  
2022年1月開催の「第四回 #作画王グランプリ ～冬の陣～」、2022年4月開催の「第5回 #作画王グランプリ ～春の陣～」で運用され、累計約260ユーザー(応募していないユーザー含む)、約240作品が応募されました。

- フロント: Next.js + Chakra-UI (TypeScript)
- バックエンド: Frourio + Prisma (TypeScript)
- 開発・運用時期: 2021年末～2022年初旬
- 実装期間: 約3週間

### 作画王グランプリ エントリー審査システム V2

システムv1の実装をベースに、Twitter OAuth V2 への移行や自動集計システムを組み込んだシステムv2を開発しました。  
システムv2はバンドリ！と音声創作ソフト「CeVIO AI」のコラボレーションプロジェクト「夢ノ結唱」、みんなで遊べるストーリーエンタテインメントプラットフォーム「monogatary.com」、超新星のアニメーターを決める作画のバトルロワイヤル「作画王グランプリ」の3つのコラボレーションによって2023年1月に開催された「夢ノ結唱×作画王グランプリ コラボコンテスト」で運用され、約100作品が応募されました。

- フロント: Next.js + Chakra-UI (TypeScript)
- バックエンド: Blitz.js + Prisma (TypeScript)
- 開発・運用時期: 2022年末～2023年初旬
- 実装期間: 約2カ月
